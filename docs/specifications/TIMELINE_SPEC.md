# タイムライン仕様書 (Timeline Specification)

## 概要

タイムラインビューは、タスクをガントチャート形式で表示し、時系列での進捗管理を可能にします。

## 2. 機能仕様

### 2.1 表示

**タスクバー**:
- 開始日〜期限日を横棒で表示
- 進捗率を色で表現
- ホバーで詳細情報表示

**タイムスケール**:
- 日/週/月単位で切り替え
- ズームイン/アウト

### 2.2 編集

**ドラッグ&ドロップ**:
- バーをドラッグして期間変更
- 左右の端をドラッグして開始日/期限日変更

**依存関係**:
- タスク間の依存関係を矢印で表示
- ドラッグで依存関係を設定
