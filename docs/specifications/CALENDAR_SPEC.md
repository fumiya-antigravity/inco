# カレンダー仕様書 (Calendar Specification)

## 概要

カレンダービューは、タスクをカレンダー形式で表示し、期限日ベースでの管理を可能にします。

## 2. 機能仕様

### 2.1 表示

**月表示**:
- 1ヶ月分のカレンダー
- 各日にタスクを表示
- 期限日でグループ化

**週表示**:
- 1週間分の詳細表示
- 時間軸も表示可能

**日表示**:
- 1日分の詳細表示
- タイムライン形式

### 2.2 タスク表示

**カレンダーセル**:
- タスクキー + タスク名
- 優先度で色分け
- 最大3件まで表示、それ以上は「+2件」と表示

**クリック挙動**:
- タスククリックで詳細パネル表示
- 日付クリックでその日のタスク作成

### 2.3 ドラッグ&ドロップ

**機能**:
- タスクをドラッグして別の日に移動
- ドロップ時に期限日を自動更新
